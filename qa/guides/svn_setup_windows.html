SVN Setup (Windows)

Purpose: This guide is meant to help get SVN setup for those of you using Windows since it does not have native support.
Steps
  1. Download and install MobaXterm.
  2. After install, open it and click on the “Packages” option in the menu.
  3. Search svn in the search bar and install git-svn.
  4. After git-svn installs, start a local terminal session in MobaXterm and setup using this guide.
Note:
By default, your group2repo directory will be located under “C:\Users\Username\Documents\MobaXterm\home”.
